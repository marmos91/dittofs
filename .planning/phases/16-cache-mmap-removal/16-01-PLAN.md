<objective>
Add a hash-keyed read surface `Get(ctx, hash) ([]byte, error)` to the `pkg/blockstore/local.LocalStore` interface and implement it on both backends (`*FSStore` as a one-line delegate over the existing `ReadChunk`; `*MemoryStore` as an interface-satisfier stub returning `ErrChunkNotFound`).

Purpose: Phase 16's cache rewire (Plan 02) needs a single content-addressed local read primitive. The signature is chosen for byte-for-byte forward compatibility with Phase 17's unified `BlockStore.Get` — the engine call site narrows the receiver type later without renaming (D-01).

Output: Extended `LocalStore` interface with one new method, two implementations, and a conformance assertion in `localtest/suite.go`.
</objective>

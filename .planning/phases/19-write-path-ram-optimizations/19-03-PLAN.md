    - `func newGroupCommit(fsyncFn func() error) *groupCommit`.
    - `func (g *groupCommit) Sync(ctx context.Context) error`:
      - allocate `ch := make(chan error, 1)` (cap=1 to absorb send if caller already returned via ctx.Done).

    - `func (g *groupCommit) fire()`:
      - Lock g.mu; swap-out pending into local waiters slice; clear timer; unlock.
      - Call fsyncFn(); broadcast result to every waiter via ch <- err.

      var ErrTruncated = errors.New("backup: envelope truncated")
      var ErrCRCMismatch = errors.New("backup: CRC32 verification failed")

    Exported helpers:
    - Writer type: wraps an io.Writer with a running CRC32 hash via io.MultiWriter pattern
      NewWriter(w io.Writer, engineTag string) (*Writer, error) -- writes magic, version, engine_len, engine_tag to w; returns Writer that accumulates CRC on all subsequent writes
      (*Writer).Write(p []byte) (int, error) -- delegates to underlying writer + CRC accumulator
      (*Writer).Finish() error -- writes trailing 4-byte CRC32 to underlying writer

    - Reader type: wraps an io.Reader with a running CRC32 hash via io.TeeReader pattern
      ReadHeader(r io.Reader) (engineTag string, payloadReader io.Reader, err error) -- reads and validates magic, version, engine_len, engine_tag; returns an io.Reader that accumulates CRC on reads; error is ErrBadMagic/ErrUnsupportedVersion/ErrTruncated
      VerifyCRC(r io.Reader, accumulated hash.Hash32) error -- reads trailing 4 bytes as expected CRC, compares to accumulated.Sum32(); returns ErrCRCMismatch or ErrTruncated

    - VerifyEngine(got, want string) error -- returns ErrEngineMismatch wrapping both tags if got != want

Orchestration steps (D-24-09, strictly sequential â€” no goroutines):
  1. precheck: share exists + Enabled=false (else ErrShareEnabled); snapshot exists; snap.State == StateReady (else ErrSnapshotStateConflict via Phase 23 wrap); snap.RemoteDurable OR opts.AllowNonDurable (else ErrSnapshotNotDurable)
  2. pre-verify: ReadManifest(ManifestPath) -> VerifyRemoteDurability(ctx, remote, manifest, concurrency); fail-fast before any destructive op
  3. safety snapshot: CreateSnapshot(ctx, shareName, CreateSnapshotOpts{}) (default NoSyncGate=false per D-24-04) -> WaitForSnapshot(ctx, shareName, safetyID); assert final State == StateReady; failure wraps ErrRestoreSafetySnapFailed
  4. open dump: os.Open(MetadataDumpPath(localStoreDir, snapID)); ENOENT -> ErrSnapshotMetadataDumpMissing
  5. Reset: metaStore.(metadata.Resetable).Reset(ctx); type-assert miss -> ErrMetadataStoreNotResetable; Reset error -> wraps ErrRestoreAborted with safety snap ID in message
  6. Restore: metaStore.(metadata.Backupable).Restore(ctx, dumpFile); error -> wraps ErrRestoreAborted with safety snap ID in message
  7. post-verify: HashSetFromMetadataStore(ctx, metaStore) -> VerifyRemoteDurability(ctx, remote, hashes, concurrency); fail -> wraps ErrRestoreVerifyFailed with safety snap ID in message
  8. return nil; share STAYS DISABLED (D-24-01); operator runs `dfsctl share enable` after inspection

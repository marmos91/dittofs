Assert: the first Flush call synced exactly the original 3 (not hX, hY). Run a second bs.Flush. Assert: hX + hY are now synced.

      Write a file containing a unique hash H. Flush so H is in remote + syncedHashStore. Delete the file via bs.Delete (refcount → 0). Assert syncedHashStore.IsSynced(H) == false. Assert local CAS chunk for H is gone (local.Has(H) == false or local.Get returns ErrChunkNotFound).

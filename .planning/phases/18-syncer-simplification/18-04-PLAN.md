      func (bc *FSStore) ListUnsynced(ctx context.Context) iter.Seq2[blockstore.ContentHash, error]

    Body structure (per 18-PATTERNS.md section 10):
    1. Return a closure func(yield func(blockstore.ContentHash, error) bool).
    2. Nil-syncedHashStore early-out: if bc.syncedHashStore == nil, immediately return (yields nothing) — strict-subset invariant.
    3. Snapshot phase: declare local snapshot []blockstore.ContentHash. Call bc.Walk(ctx, func(hash, _) error { snapshot = append(snapshot, hash); return nil }). If Walk returns error, yield once with (zero ContentHash, walkErr) and return.
    4. Filter phase: for each h in snapshot: check ctx.Err() and yield (zero, err) + return on cancellation; call bc.syncedHashStore.IsSynced(ctx, h). On lookup error, yield (h, fmt.Errorf("synced lookup %s: %w", h, err)) — if yield returns false, return; else continue to next h. If synced == true, continue (skip). If synced == false, yield (h, nil) — if yield returns false, return.

    Per-hash IsSynced cost is O(1) (D-06): hash-keyed lookup is constant-time in all 3 backends.

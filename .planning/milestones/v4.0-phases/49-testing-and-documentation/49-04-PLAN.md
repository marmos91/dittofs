1. Create `pkg/bench/workload_cache_tiers.go` implementing the 6-step cache-tiers benchmark:

   The benchmark runner needs an authenticated API client (for cache evict/stats) plus a mount point (for NFS/SMB file I/O). It accepts a share name to scope cache operations.

   ```go
   type CacheTiersBenchmark struct {
       mountPoint string
       shareName  string
       client     *apiclient.Client
       fileSizes  []int64 // default: 10MB, 100MB, 1GB
   }
   ```

   For each file size, run the 6-step workflow:

   **Step 1 - Write**: Create a test file of the given size via NFS mount (sequential write). Record write throughput.

   **Step 2 - Evict all cache**: Call `client.CacheEvictForShare(ctx, shareName, &CacheEvictRequest{})` to evict both L1 and local blocks. This forces the next read to go to remote store.

   **Step 3 - Cold read**: Read the entire file via NFS mount. Measure throughput (MB/s) and latency (time to first byte, total time). This represents remote store (S3) latency. Record cache stats after read (should show L1 hits from the read filling cache).

   **Step 4 - Warm read**: Read the same file again. Now data is in L1 + local cache. Measure throughput and latency. Record cache stats (should show high L1 hit rate).

   **Step 5 - Evict L1 only**: Call `client.CacheEvictForShare(ctx, shareName, &CacheEvictRequest{L1Only: true})`. This clears memory cache but keeps local FS blocks.

   **Step 6 - L2-only read**: Read the file again. Data comes from local FS cache (not memory L1, not remote). Measure throughput and latency. Record cache stats.

   After each read step (3, 4, 6), fetch and display cache stats via `client.CacheStatsForShare()`.

   Collect results into a `CacheTiersResult` struct with per-size, per-step metrics:
   ```go
   type CacheTiersResult struct {
       Results []CacheTiersSizeResult
   }
   type CacheTiersSizeResult struct {
       FileSize      int64
       WriteStats    IOStats  // throughput, duration
       ColdReadStats IOStats  // throughput, p50, p99, cache stats
       WarmReadStats IOStats  // throughput, p50, p99, cache stats
       L2OnlyStats   IOStats  // throughput, p50, p99, cache stats
   }
   type IOStats struct {
       ThroughputMBps float64
       DurationMs     int64
       L1HitRate      float64 // from cache stats
   }
   ```
